# 2023 HL 만도 & HL 클레무브 자율주행 모빌리티 경진대회 VLF (대학)
최종: 장려상(4등)

# 대회 소개
![image](https://github.com/powerslam/KookminVLF/assets/97532643/6e23b6ca-5871-4832-b745-f7625c102ff8)

# 팀원
* 이강욱 (나)
* 이상혁

# 대회 미션
* 라인 트레이싱
* 정적 장애물 회피
* 미로 구간 탈출

# 주요 코드
* VLF      : 초음파, 서보모터, DC 모터 초기화 코드
* LineScan : 라인 카메라 초기화 및 전처리 코드
* fast     : 속도 빠른 코드
* narrow   : 길이 좁을 때 코드
* wide     : 길이 넓을 때 코드
